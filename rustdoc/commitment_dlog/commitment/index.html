Module commitment_dlog::commitment

source · [−]
Expand description

This module implements Dlog-based polynomial commitment schema. The folowing functionality is implemented

  1. Commit to polynomial with its max degree
  2. Open polynomial commitment batch at the given evaluation point and scaling factor scalar producing the batched opening proof
  3. Verify batch of batched opening proofs

Note that if one of the polynomial comes specified with a degree bound, the evaluation for the last segment is potentially shifted to meet the proof.
